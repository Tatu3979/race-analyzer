# Race Analyzer

FIT ファイルを区間分析し、次のレース戦略の AI 相談用プロンプトを生成する Web アプリ。

## 特徴

- **アップロードしたデータは外部に一切送信されません。** FIT ファイルもフォーム入力もすべてブラウザ内で処理され、サーバーには送られません。
- アカウント登録不要。URL を開けばすぐ使えます。
- 区間（200m / 500m / 1km）ごとにペース・ピッチ・ストライドを可視化。
- レース情報と現在のトレーニング状況を入力すると、Stage 1（弱点特定）→ Stage 2（手法選択）→ Stage 3（プラン作成）の3段階プロンプトを生成し、普段使う AI チャットにコピペして相談できます。

## 使い方

公開URL: https://race-analyzer-2026.web.app

1. レースの FIT ファイル（Garmin Connect → アクティビティ → エクスポート → オリジナル）を画面にドラッグ＆ドロップ、またはクリックして選択。FIT がない場合は手動でラップを入力することもできます。
2. ラップ一覧と区間分析グラフが表示されます。気になるラップを選ぶとそのラップ内にズームインし、「全体に戻る」でアクティビティ全体表示に戻れます。
3. 画面下のフォームに、次のレースの目標距離・タイム・期日・現在の月間距離・最長走行距離などを入力します。必須項目が揃うとプロンプトが表示されます。
4. 生成された Stage 1 / 2 / 3 のプロンプトを順にコピーし、Gemini など普段使う AI に貼り付けて相談します。

## 現バージョン (v0.1.0) の制約と注意事項

- **コースの起伏は分析に含まれません。** 平坦コース前提のペース・ピッチ・ストライドの数値分析です。獲得標高や勾配補正は未実装です。
- **推奨AIは Gemini です。** ChatGPT / Claude 等でも動作しますが、生成プロンプトは Gemini で最も意図通りの出力が得られるよう設計されています。
- **リロード・タブを閉じるとデータは消えます。** アップロードした FIT、入力フォーム、生成プロンプトは保存されず、再開には再アップロードが必要です。
- **対象はランニングの単発レース・ワークアウトです。** インターバル走の詳細分析、トライアスロン、ロードバイクなど他種目のアクティビティは現時点で対応していません。
- **アカウント機能・履歴保存・過去レース比較・複数FIT同時アップロードはありません。**
- **Garmin / Strava / Coros 等のサービス連携はありません。** 各サービスから FIT ファイルを手元にダウンロードして使う形式です。

## 技術スタック

- Vite + React + TypeScript
- fit-file-parser（FITパース）
- Recharts（グラフ描画）
- Firebase Hosting

使用しているOSSの一覧と各ライセンスは [THIRD_PARTY_NOTICES.md](./THIRD_PARTY_NOTICES.md) を参照してください。

## 開発の経緯

市民ランナーが自分の走行データを AI に相談する際の「前提情報の入力が毎回煩雑」「個人の生体データを外部に送りたくない」という2つの課題から出発したアプリ。FIT の数値解析と、AI に渡すための構造化プロンプト生成をすべてブラウザ内で完結させている。開発過程と設計判断の詳細は note 記事 (note記事URL) に記録している。
